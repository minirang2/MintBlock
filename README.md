# 🍀MintBlock
서울민트초코가 만든 엔트리 비공식 블록
<br><br>
사용법: 
1. 엔트리 작품만들기 페이지
2. F12 ==> 콘솔
3. 
```javascript
$.get('https://raw.githubusercontent.com/minirang/MintBlock/refs/heads/main/block.js')
```
<br><br>
참고로 스페셜 블록 아주 일부 참고했습니다.
<br><br>
소스코드를 마음껏 참고해도 되나 똑같이
따라하는것은 허용되지 않습니다.
<br><br>
코드 자체는 자바스크립트 코드 실행 블록에 있는 eval 빼고 다 안전합니다.
버그는 신속히 제보 바랍니다
<br><br>
